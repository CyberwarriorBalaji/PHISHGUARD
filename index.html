HOW IT WORKS
🌐

URL Scanner

Detects phishing via TLD analysis, lookalike leet-speak characters, brand impersonation, suspicious keywords, URL shorteners, raw IP addresses, and HTTP vs HTTPS.

📄

File Checker

Checks extension type, reads magic bytes (real file type), double-extension tricks (photo.jpg.exe), embedded scripts, file size anomalies, and computes SHA-256 hash.

📱

APK Analyzer

Parses APK (ZIP) structure, reads AndroidManifest, detects dangerous permissions (SMS, Camera, Root), package name spoofing, unsigned APKs, and suspicious code strings.

🔐

Hash Lookup

CRC32, SHA-1, and SHA-256 hashes are computed in your browser. Paste the SHA-256 on VirusTotal.com to cross-check against 70+ antivirus engines instantly.

📊

Risk Scoring

Each suspicious signal adds to a 0–100 risk score. 0–25 = Safe ✅, 26–55 = Warning ⚠️, 56–100 = Danger 🚨. Severity-weighted for accuracy.

🔒

100% Private

Nothing is sent to any server. URL analysis is regex/heuristic-based. File and APK analysis uses the browser's FileReader and Web Crypto APIs entirely locally.
